## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/matidr/chat_app.git
   cd chat_app
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
